
Choice-Based Calculator

Welcome to the Choice-Based Calculator repository! This calculator provides users with the ability to perform various arithmetic operations through a simple command-line interface.

Features
Choose from a variety of arithmetic operations.
Easy-to-use interface for performing calculations.
Supports addition, subtraction, multiplication, and division.
Error handling for invalid inputs.

how to Use :
1 . Clone this repository to your local machine:

git clone <repository_url>

2 . Navigate to the directory of the cloned repository:

cd choice-based-calculator-java

3 . Compile the Java files:

Calculator.java

4 . Run the calculator 

5 . Follow the on-screen prompts to perform calculations.

  EXAMPLE

Welcome to the Choice-Based Calculator!

Entre the numbers : 
10
5

press 1 to Add
press 2 toSubtract
press 3 toMultiply
press 4 toDivide

Enter your choice: 1



Result: 10 + 5 = 15
